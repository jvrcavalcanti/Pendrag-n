# Micro Framework MVC PHP

Lincense MIT

## Start

```bash
composer install
```

## Scripts

### Serve
```bash
composer serve
```

### Migrate
```bash
composer migrate
```

### Migrate Rollback
```bash
composer migrate.rollback
```

### Migrate Refresh
```bash
composer migrate.refresh
```

### Clear Images
```bash
composer clear.images
```

### Tests
```bash
composer test
```

### Make Migration
```bash
composer make.migration TestTable
```

### Make Model
```bash
composer make.model User
```

### Make Controller
```bash
composer make.controller UserController
```

### Make Middleware
```bash
composer make.middleware Auth
```
